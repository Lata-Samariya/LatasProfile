Projects Made

ECG Signal Processing and Classification Using Neural Networks

In this project, I processed normal and abnormal ECG signals from the MIT-BIH Arrhythmia Database to classify them based on extracted features. The steps included: Noise Addition: Random noise was added to both ECG signals to simulate real-world data imperfections. Noise Filtering: A Butterworth high-pass filter was applied to the noisy signals to eliminate low-frequency noise, allowing the main ECG signal characteristics to remain prominent. Feature Extraction: Statistical features, such as mean, standard deviation, skewness, kurtosis, maximum, and minimum values, were extracted from the filtered signals. These features provided insights into the variations between normal and abnormal ECG patterns. Classification with Neural Network: The extracted features were then fed into a neural network classifier. The network was trained to distinguish between normal and abnormal ECGs based on these features. The output of this classification process enables the identification of abnormal ECG patterns, aiding in the detection of arrhythmias and other heart conditions. The tabulated features provide a structured comparison, highlighting the signal differences that inform classification decisions.
